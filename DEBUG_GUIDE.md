## 获取错误日志的步骤

### 方法 1：开发者工具（推荐）

1. **打开应用**
   ```bash
   npm run electron:serve
   ```

2. **打开开发者工具**
   - 按 `F12` 键
   - 或按 `Ctrl+Shift+I`

3. **查看错误信息**
   - 点击 `Console` 标签
   - 查找红色的错误信息
   - 截图或复制所有错误

4. **查看网络请求**（如果有的话）
   - 点击 `Network` 标签
   - 查看是否有失败的请求（红色的）

5. **检查元素**
   - 点击 `Elements` 标签
   - 查看搜索框的 HTML 结构和样式

### 方法 2：终端日志

运行开发模式时，终端也会显示错误：
- 查找终端中的 `ERROR` 或 `Warning` 信息
- 复制相关的错误堆栈

### 需要的信息

请提供以下任意一项：
1. 开发者工具 Console 的截图或错误文本
2. 终端的错误信息
3. 搜索框空白的截图
